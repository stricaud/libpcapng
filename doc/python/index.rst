libpcapng Python — pcapsh binding
==================================

Execute ``.pcapsh`` scripts from Python and receive raw Ethernet frames
for each :func:`wrpcap` call. No file I/O required — packets come back
as :class:`bytes` objects or through a per-packet callback.
